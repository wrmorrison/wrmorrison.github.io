Mahalo! Workshop Follow-up

Thank you all for attending! It was a real pleasure to have each and every one of you there. Please note that two Photoshop preferences were set on the workshop computers that may influence your ability to replicate the exercises on your own:

  1. Preferences > Interface > Uncheck Open Documents as Tabs
  2. Preferences > Cursors > Full Size Brush Tip

Don't forget to do your homework:

  1. Study the Refine Edge and Mask Edge commands
  2. Study the Color Range command
  3. Study Quick Selection Tool use
  4. Study Smart Objects and Smart Filter use

 

1. Open the image to be masked ...

  1. Open diamond-head.jpg in Adobe Photoshop (image right)
  2. Tap the letter 'f' so you're working on a solid grey background
    • Note: Tapping 'f' a second time gives you a black background
    • Note: Tapping 'f' a third time takes you back to normal viewing mode
  3. From the Menu bar, go to File > Save As...
  4. Save your file as a .psd (download ZIPPED .psd of this step)
    • a .psd (Photoshop Document) will preserve ALL your layers and settings

2. Select the sky ...

  1. In the Tool bar, select the Magic Wand Tool (image right)
  2. In the Options bar, reset the Magic Wand Tool
    • To reset a tool, right-click on the tool icon in Options bar (left end) and choose Reset Tool
  3. With the Magic Wand Tool, click the sky to create an initial selection
  4. To select the remaining portions of the sky, hold down the SHIFT key
    and click the un-selected areas
    • Note: When you hold the SHIFT key down, a little + is added to the Magic Wand cursor
    • Note: If you need to start a new selection, from the Menu bar go to Select > Deselect
  5. When your final selection is in place, select the Move Tool from the Tool bar to avoid accidental selection or de-selection issues
    • Reality check: This is a VERY easy selection exercise because 1.) Diamond Head is a relatively hard edged object and 2.) there is a clear difference in contrast between Diamond Head and the sky

3. Mask and invert ...

  1. Open the MASKS panel
  2. Click the [Add a pixel mask] button (image right)
    • Note: You have created a mask based on a selection
    • Note: The mask created is doing the exact opposite of what we want (Diamond Head is hidden - the sky is visible)
  3. In the MASKS panel, click the [Invert] button
    • Note: The mask is now doing exactly what we want (Diamond Head is visible - the sky hidden)
    • Note: Transparency in Photoshop is represented as a checkerboard pattern
  4. Save your file as a .psd (download ZIPPED .psd of this step)
  5. Tap the letter 'f' until you return to normal screen mode

4. Open a second image ...

  1. Open chopper.jpg in Adobe Photoshop (image right)

5. Move and re-stack ...

  1. With the Move Tool, drag the chopper.jpg on top of the diamond-head.jpg
    • Note: Adding SHIFT after you begin dragging, and letting SHIFT go after you drop, aligns the chopper.jpg to the center of diamond-head.jpg
  2. If necessary, use the Move Tool to align the two layers
  3. In the Layers Panel, drag the chopper layer below the Diamond Head layer (image right)
  4. Save your file (download ZIPPED .psd of this step)
  5. Close the chopper.jpg image (do not save)
  6. Tap the letter 'f' so you're working on a solid grey background

6. Refine the mask edge ...

  1. In the Layers Panel, select the mask attached to the Diamond Head layer
    • Note: There are 3 selectable thumbnails in the Layers panel
    • Note: When masking it is critical to be aware of which thumbnail is selected. Many masking errors and issues result from a user being unaware of which thumbnail is highlighted in the Layers panel
  2. Open the MASKS panel
  3. Click the [Mask Edge] button (image right)
    • Note: Clicking the [Mask Edge] button opens the Refine Mask dialogue box where you can fine-tune your mask
    • Note: The Refine Mask dialogue box is very useful and powerful. You should definitely learn more about this feature (study at home)
  4. In the Refine Mask dialogue box, set the following:
    • View Mode: On Layers (so we can see changes happen to the image)
    • Output: Layer Mask (so changes are applied to our mask)
  5. Select the Zoom Tool in the Refine Mask dialogue box
  6. Zoom in to the plants silhouetted against the sky along the edge of Diamond Head
    • Note: There is a less than ideal mask around the silhouetted plants - we will improve this
  7. In the Edge Detection section:
    • For Radius, experiment by increasing the value (slider right) - you should be watching your silhouetted plants as you do this
    • Note: By raising this value you are asking Photoshop to go out in search of a better edge
  8. Experiment with other sliders including Contrast and Shift Edge until you believe that you've improved your mask edge
  9. Save your file (download ZIPPED .psd of this step)

7. Adjust the composite ...

  1. In the Layers Panel, select the Diamond Head layer thumbnail
  2. Open the ADJUSTMENTS panel
  3. Click on the Create a new Black & White adjustment layer icon (image right)
    • Note the effect: The adjustment layer was added above the Diamond Head layer
    • Note: Added layers always come in above the layer currently selected
    • Note: Photoshop gives you a FREE mask when you add a fill or adjustment layer
  4. In the ADJUSTMENTS panel, experiment with the black & white adjustments available until you get the look you want
  5. Save your file (download ZIPPED .psd of this step)

8. Mask the adjustment ...

  1. Select the Brush Tool from the Tool bar (image right)
  2. In the Options bar, reset the Brush Tool
    • To reset a tool, right-click on the tool icon in Options bar (left end) and choose Reset Tool
  3. With the Brush tool, right-click on your image and set the following:
    • Size: 6px
    • Hardness: 100%
  4. At the bottom of the toolbar, click the Default Foreground and Background Color icon (the smaller pair of square color chips)
  5. Click the double arrow switcher icon so that black is the foreground color (the top square of the larger pair of square color chips)
  6. Zoom in to the man standing on the top of Diamond Head
  7. In the Layers Panel, select the mask attached to the Black & White adjustment layer
    • Note: There are 5 selectable thumbnails in the Layers panel
    • Note: When masking it is critical to be aware of which thumbnail is selected. Many masking errors and issues result from a user being unaware of which thumbnail is highlighted in the Layers panel
  8. Painting with black, paint over the man to hide (conceal) the effect of the Black & White adjustment layer
  9. Save your file (download ZIPPED .psd of this step)
  10. Tap the letter 'f' until you return to normal screen mode
