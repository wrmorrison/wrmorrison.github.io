Exercise 2: Masking with the Gradient Tool

Understanding how the leverage the unique power of the Gradient Tool in masking is vital for any Photoshop user. Knowing how to blend images together not only delivers a professional look, it’s also quick and easy to achieve. In this exercise we will explore and witness the effects of masking different styles of the Gradient Tool.

You will know how to:

  1. Create a blank mask
  2. Modify a mask with the Gradient Tool
  3. Toggle the visibility of a mask on/off (Enable/Disable)
  4. View a mask
  5. Delete a mask
  6. and fill a mask

1. Opening the image to be masked ...

  1. Open eye.jpg in Adobe Photoshop (image right)
    • Source (and thanks to): Face Paint by Alice Birkin
  2. Tap the letter 'f' so you're working on a solid grey background
    • Note: Tapping 'f' a second time gives you a black background
    • Note: Tapping 'f' a third time takes you back to normal viewing mode
  3. From the Menu Bar, go to File > Save As...
  4. Save your file as a .psd (download ZIPPED .psd of this step)
    • a .psd (Photoshop Document) will preserve ALL your layers and settings

2. Add a pixel mask …

  1. Open the MASKS panel
  2. Click the [Add a pixel mask] button (image right)
    • Note: By default, a white (inert) mask is added
  3. Save your file (download ZIPPED .psd of this step)

 

3. Add a solid color fill layer ...

  1. At the bottom of the LAYERS panel, click the ‘Create new fill or adjustment layer’ icon (it looks like a black & white cookie)
  2. From the options available, choose ‘Solid Color…’
  3. In the Pick a solid color: dialogue box, select black as the fill color and hit OK
    • Note the effect: The solid color fill layer was added above the eye layer
    • Note: Added layers always come in above the layer currently selected
    • Note: Photoshop gives you a FREE mask when you add a fill or adjustment layer
  4. In the LAYERS panel, drag the solid color layer below the eye layer
  5. Save your file (download ZIPPED .psd of this step)

4. Select and set the gradient tool …

  1. In the Tool bar, select the Gradient Tool (image right)
  2. In the Options bar, reset the Gradient Tool
    • To reset a tool, right-click on the tool icon in Options bar (left end) and choose Reset Tool
  3. In the Options bar, select the ‘Black, White’ gradient preset
  4. For gradient style, choose the Linear Gradient

5. Mask with a linear gradient …

  1. In the Layers Panel, select the mask attached to the eye layer
    • Note: There are 4 selectable thumbnails in the Layers panel
    • Note: When masking it is critical to be aware of which thumbnail is selected. Many masking errors and issues result from a user being unaware of which thumbnail is highlighted in the Layers panel
  2. With the Gradient Tool, click-hold then drag-release a gradient across your image from right to left (image right)
    • Note: You have created a gradient mask
  3. Drag out another gradient – notice that this second gradient replaces your previous one
  4. Experiment by dragging in different directions
  5. Experiment by adjusting the drag-release distance
  6. Save your file (download ZIPPED .psd of this step)

6. Some masking nuances …

  1. To Toggle the visibility of a mask on/off:
    • Click the Disable/Enable button (little eye) at the bottom of the MASKS panel
  2. To view a mask: (image right)
    • On a Mac, Option-click directly on the mask thumbnail
    • On a PC: Alt-click directly on the mask thumbnail
    • Click the layer thumbnail to return to normal viewing
  3. To reset a mask to default white:
    • DEMO: Delete and create a new mask
    • Select the mask attached to the eye layer. From the Menu bar, go to Edit > Fill. Choose white as the fill color
  4. Demo: Masking the solid color fill layer

7. Mask with a radial gradient ...

  1. In the tool bar, select the Gradient Tool
  2. For gradient style, choose the Radial Gradient
  3. Click-hold and drag outward from the center of the eye - then release
    • Note: If the gradient mask created is the opposite of what you feel is intuitive, select the Reverse check box in the Options bar
  4. Experiment by dragging a few additional radial gradients (image right)
  5. Save your file (download ZIPPED .psd of this step)
  6. Tap the letter 'f' until you return to normal screen mode

8. Open a second image ...

  1. Open leaves.jpg in Adobe Photoshop
    • Source (and thanks to): Colorful Autumn Leaves by Vera Kratochvil

9. Move and re-stack ...

  1. With the Move Tool, drag the leaves.jpg on top of the eye.jpg
    • Note: Adding SHIFT after you begin dragging, and letting SHIFT go after you drop, aligns the leaves.jpg to the center of eye.jpg
    • Reality check: Pixel counts
  2. If necessary, use the Move Tool to align the two layers
  3. In the Layers Panel, drag the leaves layer below the eye layer (image right)
  4. Save your file (download ZIPPED .psd of this step)
  5. Close the leaves.jpg image (do not save)
  6. Tap the letter 'f' so you're working on a solid grey background

10. Experiment with different gradients ...

  1. In the Layers Panel, select the mask attached to the eye layer
    • Note: There are 5 selectable thumbnails in the Layers panel
    • Note: When masking it is critical to be aware of which thumbnail is selected. Many masking errors and issues result from a user being unaware of which thumbnail is highlighted in the Layers panel
  2. Experiment by blending the two image layer together with the linear, radial, angled, reflected, and diamond styles (image right)
    • Note: As noted above, some gradient styles will draw more intuitively if you click the Reverse check box in the Options bar
  3. Save your file (download ZIPPED .psd of this step)

11. Adjust the leaves ...

  1. In the Layers Panel, select the leaves layer thumbnail
  2. Open the ADJUSTMENTS panel
  3. Click on the Create a new Black & White adjustment layer icon (image right)
    • Note: The adjustment layer was added above the leaves layer
    • Note: Added layers always come in above the layer currently selected
    • Note: Photoshop gives you a FREE mask when you add a fill or adjustment layer
  4. In the ADJUSTMENTS panel, experiment with the various Black & White adjustment sliders
  5. Save your file (download ZIPPED .psd of this step)

12. Mask the adjustment ...

  1. In the Layers Panel, select the mask attached to the black & white adjustment layer
    • Note: There are 5 selectable thumbnails in the Layers panel
    • Note: When masking it is critical to be aware of which thumbnail is selected. Many masking errors and issues result from a user being unaware of which thumbnail is highlighted in the Layers panel
  2. Experiment masking the black & white adjustment with the linear, radial, angled, reflected, and diamond styles (image right)
  3. Save your file (download ZIPPED .psd of this step)
  4. Tap the letter 'f' until you return to normal screen mode
